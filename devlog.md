# DEVLOG: AI 수익 자동화 파이프라인 만들기

## 작업 일시
- 시작: 2026-06-01T18:04
- 완료: 2026-06-01T18:16

## 완료한 단계

### 1단계: 프로젝트 설정 및 라이브러리 설치
npm init으로 프로젝트를 초기화하고 express, openai 패키지를 설치했다.

### 2단계: SEO 블로그 콘텐츠 뼈대 스크립트 작성
재테크 노하우와 직장인 번아웃 극복 주제로 GPT를 활용해 SEO 최적화 블로그 글을 생성하는 blogGenerator.js를 작성했다.

### 3단계: 디자인 자동화를 위한 프롬프트 생성기 추가
부와 성공을 연상시키는 프리미엄 골드 럭셔리 스타일의 미드저니 프롬프트 생성 함수를 blogGenerator.js에 추가했다.

### 4단계: 챗봇 서버 기본 구조 만들기
Express를 사용해 포트 3000번에서 동작하는 기본 챗봇 서버(server.js)를 작성했다.

### 5단계: 배송 조회 인텐트(Intent) 라우트 추가
사용자 메시지에서 배송 관련 키워드를 감지해 적절한 응답을 반환하는 POST /webhook 라우트를 server.js에 추가했다.

### 6단계: 전체 파이프라인 테스트 및 결과 확인
서버를 실행하고 curl로 배송 조회 API를 테스트하여 인텐트가 정확히 분류되고 응답이 정상 반환되는 것을 확인했다.
