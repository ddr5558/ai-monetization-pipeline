# CLAUDE.md — AI 수익 자동화 파이프라인

## 토큰 최적화 지침 (반드시 먼저 따를 것)

1. **코드 구조 파악은 `KNOWLEDGE.md`를 먼저 읽어라.** 전체 파일을 순차적으로 탐색하지 말 것.
2. 작업에 **관련된 파일만 선별해서** 읽어라 (KNOWLEDGE.md의 "파일별 역할" 표 참고).
3. 디렉토리/파일 목록이 필요하면 전체 나열 대신 **rtk.py로 요약**해서 받아라
   (Git Bash 형식 `ls -R`를 입력으로):
   `ls -R . | grep -v node_modules | python rtk.py`
4. 이미 아는 사실을 다시 설명하거나, 읽은 파일 내용을 통째로 되풀이하지 말 것.

## 응답 스타일

- 한국어 존댓말, **간결하게.** 장황한 서론·중복 설명 금지.
- 변경한 파일과 핵심 결과만 보고. 코드 전체를 다시 출력하지 말 것.
- 추측은 추측이라고 명시. 모르면 모른다고.

## 프로젝트 개요

뉴스를 AI(Claude)로 경제·재테크 기사로 만들어 **워드프레스 + 티스토리에 하루 3회 자동 발행**하고, 글마다 **유튜브 쇼츠 대본**까지 생성하는 파이프라인. 광고(카카오 애드핏)로 수익화.

- 클라우드(GitHub Actions): 기사 생성 → 워드프레스
- 로컬(작업 스케줄러): 티스토리 미러링 + 쇼츠 대본 + 알림
- 상세 구조는 **KNOWLEDGE.md** 참고.

## 작업 규칙

- 커밋·푸시는 사용자가 요청할 때만. main 브랜치 직접 사용(이 저장소 관행).
- 비밀키 파일(`.shorts-config.json`, `.alert-config.json`, `whale-profile/`)은 절대 커밋 금지(gitignore됨).
